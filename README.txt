
# Portfolio Bundle
This folder contains a deployable static site:

- `index.html` – your portfolio
- `assets/` – placeholder images and sample MP4s (1s black clips). Replace with your real media.
- `your-resume.pdf` – (optional) add your PDF next to index.html

To Push Again:

git remote -v

git remote set-url origin https://github.com/<username>/<username>.github.io.git


git checkout -B main
git add -A
git commit -m "Site updates"
git push -u origin main
